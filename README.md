# Curriculum Vitae

Curriculum Vitae 是一个基于 Qt Widgets 和 C++ 开发的本地个人发展规划系统。系统面向大学生四年成长记录场景，支持课程、成绩、经历、成就、能力画像、职业目标规划和简历导出等功能。

## 功能特性

- 个人资料管理：维护姓名、学校、专业、发展方向和邮箱信息。
- 课程记录：记录学期、课程名称、学分、成绩和绩点。
- GPA 计算：根据课程学分和成绩自动计算加权 GPA。
- 经历管理：记录项目、竞赛、实习和课外活动经历。
- 成就管理：记录奖项、荣誉、角色和志愿服务。
- 数据持久化：支持将成长档案保存为 JSON 文件，并从 JSON 文件导入。
- 简历导出：生成 Markdown 格式个人简历。
- 成长画像分析：根据课程、经历和成就自动生成发展建议。
- 能力雷达图：基于学业基础、实践能力、竞赛科研、荣誉成果、领导协作五个维度生成个人能力画像。
- 职业目标路线图：根据目标方向计算匹配度，识别优势与短板，并生成四年行动路线图。

## 技术栈

- C++17
- Qt 6 Widgets
- qmake
- JSON 文件存储
- Qt 原生 `QPainter` 自绘雷达图

## 模块说明

- `MainWindow`：主界面控制器，负责页面切换、事件绑定、首页刷新和功能模块集成。
- `DataStore`：数据管理中心，负责数据保存读取、GPA 计算、能力评分、成长建议和职业目标分析。
- `models`：定义课程、经历、成就和个人资料等核心数据模型。
- `RadarChartWidget`：自绘能力雷达图组件，负责五维能力画像可视化。
- `ResumeExporter`：简历导出服务，负责生成 Markdown 简历文件。

## 运行方式

1. 使用 Qt Creator 打开 `curriculum vitae.pro`。
2. 选择可用的 Desktop Qt Kit。
3. 构建并运行项目。

也可以在已有构建目录中使用命令行构建：

```bash
make -C build/Qt_6_11_1_for_macOS-Debug
```

## 使用流程

1. 在首页填写或查看个人基本信息。
2. 在“课程与 GPA”页面录入课程、学分和成绩。
3. 在“项目/竞赛/实习”页面录入实践经历。
4. 在“成就与角色”页面录入奖项、荣誉、角色或志愿服务。
5. 返回首页查看 GPA、统计卡片、成长建议、能力雷达图和职业目标路线图。
6. 使用“保存数据”导出 JSON 成长档案，或使用“导入数据”恢复已有档案。
7. 在“简历导出”页面导出 Markdown 简历。

## 数据与导出

系统使用 JSON 文件保存完整成长档案，包含个人资料、课程记录、经历记录和成就记录。Markdown 简历导出内容包括个人信息、课程记录、项目/竞赛/实习经历、成就与角色、能力雷达画像、职业目标规划摘要和成长建议。

## 职业目标分析

系统内置多个目标方向：

- 数据分析师
- 软件开发工程师
- 金融科技/金融分析
- 保研/科研方向
- 产品/项目管理方向

职业目标分析基于本地规则引擎实现，不依赖联网服务。系统会结合能力雷达分数和目标关键词，计算目标匹配度，并生成优势、短板和阶段性行动建议。
